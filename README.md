# एक हराएको कलम — Nepali Blog Website

एक पेशेवर नेपाली ब्लग वेबसाइट जसमा कविता, निबन्ध र गीता व्याख्या समावेश छ।

## Website Structure

### Pages Created:

1. **index.html** - Homepage (गृहपृष्ठ)
   - Welcome message and introduction
   - Quick links to all sections (Poems, Essays, Gita)
   - Brief previews of each content type
   - Newsletter subscription form
   - Clean and professional layout

2. **poems.html** - Poetry Page (कविता)
   - Complete collection of Nepali poems
   - Individual poem pages with proper formatting
   - Poems included:
     - हाफ हाफ हाफ (Half Half Half)
     - तीजको लहर आयो बरीलै (Teej Festival)
     - हजार सुगन्धहरु माझ कुनै बागमा (Among a Thousand Fragrances)

3. **essays.html** - Essays Page (निबन्ध)
   - Thoughtful essays on philosophy and life
   - Essays included:
     - प्रकृति विरुद्धको विद्रोह (Rebellion Against Nature)
   - Well-structured with headings and sections

4. **gita.html** - Gita Commentary Page (गीता व्याख्या)
   - Simple Nepali explanations of Bhagavad Gita verses
   - Currently includes:
     - First Shloka explanation and interpretation
   - Life application and philosophical insights

## Design Features

### Responsive Design
- Mobile-first approach
- Works perfectly on phones, tablets, and desktops
- Adaptive navigation menu
- Flexible grid layouts

### Typography
- **Noto Serif Devanagari** - For body text with proper Nepali support
- **Mukta** - Clean, readable sans-serif for UI elements
- **Pridi** - Elegant serif for headings
- All fonts support Devanagari script perfectly

- Dark mode by default (easy on eyes)

- Automatic system preference detection
- Smooth transitions between themes
- Colors:

## How to Deploy to Neocities

1. Upload all HTML files to the root directory
2. Upload the `Design/` folder with style.css and script.js
3. Upload the `Images/` folder if you have images
4. Make sure all file paths are correct
5. Test all navigation links
